# Grammar data attribution

The N5, N4, and N3 grammar JSON files in this directory come from the
[Hanabira Japanese content repository](https://github.com/tristcoil/hanabira.org-japanese-content).
Hanabira publishes this content under a Creative Commons license and asks
applications using it to link back to [hanabira.org](https://hanabira.org/).

AI-Kotoba uses the source records as a grammar index and reference skeleton.
Localized explanations and practice questions shown to learners are generated
for AI-Kotoba on demand and cached per learner.

JLPT levels here are study-oriented classifications, not an official exhaustive
grammar syllabus published by the Japan Foundation.
